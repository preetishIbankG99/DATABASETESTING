CLUSTER
========
A CLUSTER IS A GROUP TABLES THAT SHARE THE SAME DATA BLOCKS
I.E,ALL THE TABLES ARE PHYSICALLY STORED TOGETHER
CREATING A CLUSTER
------------------
TO CREATE CLUSTERED TABLES.FIRST,CREATE A CLUSTER AND CREATE INDEX ON IT.
THE CREATE A TABLE IN IT.
EX::TO CREATE A CLUSTER OF DEPT AND EMP TABLES IN WHICH THE DEPTNO
WILL BE CLUSTER KEY
CLUSTER CREATING
================
SQL>CREATE CLUSTER EMP_DEPT(DEPTNO NUMBER(2));

CREATING INDEX ON CLUSTER
==========================
SQL>CREATE INDEX ON CLUSTER EMP_DEPT;

CREATING TABLE IN THE CLUSTER LINK
-----------------------------------
SQL>CREATE TABLE DEPT(DEPTNO NUMBER(2),
NAME VARCHAR2(20),
LOC VARCHAR2(20))
CLUSTER EMP_DEPT(DEPTNO);

SQL>CREATE TABLE EMP(EMPNO NUMBER(5),
NAME VARCHAR2(20),
SAL NUMBER(10,2),
DEPTNO NUMBER(2))CLUSTER EMP_DEPT(DEPTNO);

DROPPING CLUSTER
=================
SYNTAX::SQL>DROP CLUSTER<CLUSTERNAME>;
EX::DROP CLUSTER EMP_DEPT;
 IF U WANT TO DROP THE CLUSTER EVEN WHEN TABLES ARE THERE THEN GIVE THE FOLLOWING COMMAND.
EX::DROP CLUSTER EMP_DEPT
INCLUDIND TABLES
CASCADE CONSTRAINTS;
